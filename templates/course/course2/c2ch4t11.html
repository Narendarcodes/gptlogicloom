Reading Strings from Terminal

    In C, you can read strings from the terminal using various methods. Here are a few common ways to do it:

  • Using scanf:

    • Program.c

    Output:

    You entered: Hello

  • Using gets (deprecated, use with caution):

    • Program.c

    Output:
    Enter a string: Hello World
    You entered: Hello World

  • Using fgets:

    • Program.c

    Output:
    Enter a string: Hello World
    You entered: Hello World

  • Using character-wise input:

    • Program.c

    Output:
    Enter a string: Hello World
    You entered: Hello World

  • Using %[^\n]s inside scanf:
