  • In C, a function declaration provides information to the compiler about the name, return type, and parameters (if any) of a function.
  • A function declaration typically appears at the beginning of the program or in a header file.
  • Here's the syntax for a function declaration
    return_type function_name(parameters);
  • Here are a few examples of function declarations:
    • // Function declaration with no parameters and returns void
      void greetUser();
    • // Function declaration with two integer parameters and returns an int
      int addNumbers(int a, int b);
    • // Function declaration with a char pointer parameter and returns void
      void printMessage(char* message);
    • // Function declaration with no parameters and returns a double
      double getRandomNumber();
  • These declarations provide information to the compiler, allowing the program to use these functions before their actual implementations.
  • It's important to ensure that the declarations match the actual function definitions in terms of return type, function name, and parameters.<\li>
