Arithmetic Operators

  • Explanation of the types of arithmetic
  • Addition: Adds two operands.
    Example: int sum = 5 + 10; // sum is 15
  • Subtraction: Subtracts the second operand from the first.
    Example: int diff = 10 - 5; // diff is 5
  • Multiplication: Multiplies two operands.
    Example: int product = 5 * 10; // product is 50
  • Division: Divides the first operand by the second.
    Example: float quotient = 10 / 3; // quotient is 3.3333
  • Modulus (remainder): Returns the remainder when the first operand is divided by the second.
    Example: int remainder = 10 % 3; // remainder is 1
  • Increment: Increases the value of the operand by 1.
    Example: int a = 5; a++; // a is now 6
  • Decrement: Decreases the value of the operand by 1.
    Example: int b = 5; b--; // b is now 4
