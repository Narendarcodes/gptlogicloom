We'll perform operations on individual members of the structure, such as displaying book details and updating the number of pages.

  • Examples :

    Program.c


    • Structure Definition: Define a structure named Book with members title (string), author (string), year (integer), and pages (integer).
    • Function Definition: Create a function displayBook that takes a Book as a parameter and displays its details.
    • Variable Declaration: Declare a variable of type Book named myBook.
    • Input: Get input from the user for the book details.
    • Display Book Details: Print the details of the entered book using the displayBook function.
    • Update Information: Allow the user to update the number of pages for the book.
    • Display Updated Book Details: Print the updated details using the displayBook function.
    • Compilation and Execution: Compile and run the program. Enter the book details and the updated number of pages as prompted.
