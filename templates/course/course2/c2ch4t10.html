Declaring and Initializing String Variables

  • In C, strings are typically represented as arrays of characters.
  • Here are various ways to declare and initialize strings in C:
  • Using Character Array:
    char str1[] = “Hello”; // Automatically includes null terminator
  • In this example, the size of the array is automatically determined based on the length of the string “Hello,” and the null terminator \0 is included.
  • Explicit Size with Character Array:
    char str2[10] = “Hello”; // Specify size, null terminator included
  • Here, you explicitly specify the size of the character array, ensuring it is large enough to accommodate the string and the null terminator.
  • Character Array without Null Terminator:
    char str3[] = {‘H’, ‘e’, ‘l’, ‘l’, ‘o’}; // No null terminator
    This array does not include a null terminator, so it might not work with functions expecting null-terminated strings.
  • Using Pointers:
    const char *str4 = “Hello”; // Pointer to a constant string
    Here, a pointer str4 is pointing to the string “Hello.” The string is constant, so you cannot modify it through the pointer.
  • Initialization After Declaration:
    char str5[20];
    strcpy(str5, “Hello”); // Copying “Hello” to str5
    You can declare a character array and then use functions like strcpy to copy a string into it.
  • Double Quotes vs. Single Quotes:
    Strings are enclosed in double quotes, while single quotes are used for individual characters.
    For example:
    char ch = ‘A’; // Single character
    char str6[] = “Hello”; // String
