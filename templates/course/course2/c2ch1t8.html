3.flowchart to print the input number 5 times

Algorithm

  • 1. Start
  • 2. Input number a
  • 3. Now initialise c = 1
  • 4. Now we check the condition if c <= 5, goto step 5 else, goto step 7.
  • 5. Print a
  • 6. c = c + 1 and goto step 4
  • 7.Stop
