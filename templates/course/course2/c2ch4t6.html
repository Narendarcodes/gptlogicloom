Multi-Dimensional Arrays

  • In C programming, a multi-dimensional array is essentially an array of arrays.
  • It can be thought of as a table with rows and columns, where each element can be accessed using multiple indices.
  • The most common multi-dimensional array is a 2D array, but C also supports arrays with more than two dimensions.
  • In C, the syntax to declare a multi-dimensional array involves specifying the dimensions within the square brackets.
  • Here’s the general syntax:
    data_type array_name[size1][size2]…[sizeN];
  • It’s important to note that when initializing the array at the time of declaration, you can use nested curly braces to specify the initial values.
