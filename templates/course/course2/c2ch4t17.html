Table of Strings

  • To create a table of strings in C, you can use a 2D array where each row represents a string.
  • Here's a basic example:

    • Program.c

    Output:
    String 1: Hello
    String 2: World
    String 3: C Programming

  • This example creates a 2D array strings with 3 rows and a maximum string length of 19 characters.
  • The strcpy function is used to initialize each string, and then a loop is used to print the strings.
  • Adjust the size of the array and strings as needed for your specific requirements.
