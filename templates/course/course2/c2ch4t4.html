Two-Dimensional Arrays

Two-Dimensional Arrays

  • To declare a two-dimensional array in C, you specify the data type of the elements and the size of rows and columns using the syntax:
    data_type array_name[rows][columns].

Initialisation of two dimensional array

  • Initializing a two-dimensional array in C can be done at the time of declaration or later in your program.
