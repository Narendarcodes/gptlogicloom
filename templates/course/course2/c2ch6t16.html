                Factorial of 5 is 120
                    
    Explanation
    
                1. Main Function Call (`factorial(5)`)
             - Call `factorial(5)`
             - `n = 5`, not a base case, so it calls `factorial(4)`.
          
          2. First Recursive Call (`factorial(4)`)
             - Call `factorial(4)`
             - `n = 4`, not a base case, so it calls `factorial(3)`.
          
          3. Second Recursive Call (`factorial(3)`)
             - Call `factorial(3)`
             - `n = 3`, not a base case, so it calls `factorial(2)`.
          
          4. Third Recursive Call (`factorial(2)`)
             - Call `factorial(2)`
             - `n = 2`, not a base case, so it calls `factorial(1)`.
          
          5. Fourth Recursive Call (`factorial(1)`)
             - Call `factorial(1)`
             - `n = 1`, a base case, returns 1.
          
          6. Back to Third Call (`factorial(2)` returns 2 * 1)
             - `factorial(1)` result (1) is multiplied by 2, returns 2.
          
          7. Back to Second Call (`factorial(3)` returns 3 * 2)
             - `factorial(2)` result (2) is multiplied by 3, returns 6.
          
          8. Back to First Call (`factorial(4)` returns 4 * 6)
             - `factorial(3)` result (6) is multiplied by 4, returns 24.
          
          9. Back to Main Call (factorial(5) returns 5 * 24)
             - `factorial(4)`result (24) is multiplied by 5, returns 120.
          
          10. Result Displayed in main Function
              - Print: "Factorial of 5 is 120."
